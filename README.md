# ABP (Avalanche Bootstrap Percolation)
Source code (C++) for Avalanche Bootstrap Percolation (or k-core percolation)
research performed and published between 2004 and 2007 under the supervision of
Phil Duxbury at Michigan State University.


## Publications

CL Farrow, PM Duxbury, Cristian F Moukarzel (2005).\
*Culling avalanches in bootstrap percolation.*\
Physical Review E, (72), 66109.\
DOI: https://doi.org/10.1103/PhysRevE.72.066109

CL Farrow, P Shukla, & PM Duxbury (2007).\
*Dynamics of k-core percolation.*\
Journal of Physics A, (40), F581.\
DOI: https://doi.org/10.1088/1751-8113/40/27/F02
